About Us

Phone Book is a web application designed for the usage of a mobile phone user to operate with their contacts. It is a simple application which can be used to store, edit, delete and view contacts. And it is a very easy to use application. It is a very useful application for the users who are using mobile phones. It is designed based on the requirements of the users.

Technology Stack:

  • Frontend: HTML, CSS, Bootstrap, JavaScript
  • Backend: Python, Flask
  • Database:SQL
